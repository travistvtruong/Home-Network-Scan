2. Key Findings: 
All DNS queries were transmitted in plaintext over port 53, meaning that any observer on the asme network could identify applications in use, browsing behavior, and active services without decrypting any traffic

3. Recommendations: 
Port 5357 on local machine should be disabled if file sharing is uncessary on residential network
BGP exposure on router warrants monitoring
Enable DNS over HTTPS (DoH) in Brave browser

4. Conclusion:
Network segment shows a well-managed infrastructure with appropriate firewall rules. The primary finding is unencrypted DNS traffic revealing application usage, which could be mitigated with DoH
